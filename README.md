# pgr05 Workspace

This repository supports two deliverables:
- `presentation_5min/` for the in-class 5-minute policy briefing
- `assignment_4000/` for the end-of-semester 4,000-character assignment

## Structure

```text
references/
  lecture_slide/
  course_docs/
  paper/
    01_core/
    02_extended/
    auto_search/

presentation_5min/
  pt_guideline.md
  outline/
  slides/
  script/
  rehearsal/

assignment_4000/
  brief/
  outline/
  draft/
  final/
  evidence/
  references/
  review/

templates/
pre-meeting_260213.md
pre-meeting_kor_260213.md
```

## Recommended Workflow

1. Collect materials in `references/`.
2. Keep core claims in `references/paper/01_core/`.
3. Build presentation logic in `presentation_5min/`.
4. Reuse the same evidence for `assignment_4000/evidence/claim_evidence_matrix.md`.
5. Draft and revise in `assignment_4000/draft/`, then freeze final in `assignment_4000/final/`.
