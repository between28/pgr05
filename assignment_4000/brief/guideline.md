# Assignment Guideline (4000 chars)

## Goal
Write a concise, evidence-based policy assignment that is consistent with the 5-minute briefing logic.

## Constraints
- Keep one clear policy question.
- Separate current evidence from future scenario assumptions.
- Make funding/governance feasibility explicit.

## Submission Build
1. `outline/argument_outline.md`
2. `evidence/claim_evidence_matrix.md`
3. `draft/assignment_draft_v1.md`
4. `review/revision_log.md`
5. `final/assignment_final.md`
